1.	Функция Array.prototype.map какого порядка?
Функция Array.prototype.map относится к высшим порядкам, так как она принимает в качестве одного из аргументов функцию, которая будет вызвана для каждого элемента массива и вернет новый массив с трансформированными значениями.

2.	Проанализируйте код. Является ли функция lenArray высшего порядка?
const strArray = ['JavaScript', 'Python', 'PHP', 'Java', 'C'];

function mapForEach(arr, fn) {
  const newArray = [];
  for(let i = 0; i < arr.length; i++) {
    newArray.push(
      fn(arr[i])
    );
  }
  return newArray;
}
const lenArray = mapForEach(strArray, function(item) {
  return item.length;
});
// выводит [ 10, 6, 3, 4, 1 ]
console.log(lenArray)

Функция mapForEach является высшего порядка, так как она принимает функцию в качестве аргумента (fn), которая будет вызвана в цикле для каждого элемента массива. Функция lenArray не является высшего порядка, так как она принимает только обычную функцию (анонимная) в качестве аргумента. Результат работы lenArray - новый массив с длинами элементов оригинального массива, который был создан с помощью функции mapForEach.
